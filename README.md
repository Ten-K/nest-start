#### 起步

如果你还不了解nest, 请阅读[Nest中文文档](https://docs.nestjs.cn/9/introduction)。  
拉取项目后, 您可以在系统命令行工具中运行以下命令, 以启动应用程序：

``` bash
pnpm run start:dev
```

此命令启动 HTTP 服务监听定义在 src/main.ts 文件中定义的端口号。在应用程序运行后, 打开浏览器并访问 <http://localhost:3000/>。 你应该看到 Hello world! 信息。
